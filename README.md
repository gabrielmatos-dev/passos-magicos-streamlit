Passos Mágicos — Predição de Risco de Defasagem
Aplicação desenvolvida como parte do Datathon da Pós Tech Data Analytics (FIAP), utilizando dados da Associação Passos Mágicos para prever o risco de defasagem escolar de alunos.
Sobre o projeto
A Associação Passos Mágicos acompanha crianças e jovens de baixa renda por meio de indicadores educacionais e socioemocionais. Este projeto utiliza esses dados para construir um modelo preditivo capaz de identificar, com antecedência, quais alunos têm maior probabilidade de entrar em defasagem no ano seguinte.
Modelo

Algoritmo: Random Forest com calibração de probabilidade
Dados: PEDE 2022, 2023 e 2024
Avaliação: ROC-AUC de 0.80

Aplicação
A aplicação está disponível em: https://pm-predicao-risco.streamlit.app/
Tecnologias utilizadas
Python, Scikit-learn, Streamlit, Pandas, Plotly
